### 2. 关键配置
| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| Agent 名称 | 这个 Agent 叫什么？ | 根据功能自动命名 |
| 触发关键词 | 哪些词可以唤醒他？ | 根据功能推荐 |
| 绑定群聊 | 要绑定到哪个飞书群？ | 必填 |
| 执行频率 | 是一次性还是定时任务？ | 一次性 |
| 定时配置 | 如果是定时，什么时候执行？ | - |
| 消息模板 | 推送消息的格式？ | 默认格式 |
| 数据来源 | 数据从哪里获取？ | 根据功能推荐 |
| 失败处理 | 执行失败怎么办？ | 重试1次 |

### 3. 工具使用
- `openclaw cron add` - 创建定时任务
- `openclaw cron list` - 查看任务列表
- `sessions_spawn` - 创建子 Agent
- `message` - 发送消息到群聊
